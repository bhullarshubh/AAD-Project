        Q1) What does it mean when we say that an algorithm X is asymptotically more efficient than Y?
        A) X will always be a better choice for all inputs.
        B) X will always be a better choice for large inputs.
        C) X will always be a better choice for small inputs.
        D) Y will always be a better choice for small inputs.
    
            Correct answer is B. This question explains a really important concept about time complexity. We talk about large values of inputs.
            Maybe Y is a better algorithm for small inputs or sometimes X and sometimes Y but for large inputs X wins.
        
        Q2) What is time complexity of following code :

        int count = 0;
        for (int i = N; i > 0; i /= 2) {
            for (int j = 0; j < i ; j++) {
                count += 1;
            }
        }

        A) O(N * N)
        B) O(N * log N)
        C) O(N * log(log(N)))
        D) O(N)
        E) O(N * Sqrt(N))
    
            Correct answer is D. This question will teach you to think complexity in loops and we need to use mathematical techniques in few 
            questions to calculate their time complexity.
        
        Q3) What is the time complexity of the following code :

        int i, j, k = 0;
        for (i  = n/2; i <= n; i++) {
            for (j = 2; j <= n; j = j * 2) {
                k = k + n/2;
            }
        }

        A) Θ(n)
        B) Θ(nLogn)
        C) Θ(n^2)
        D) Θ(n^2 / Logn)
        E) Θ(n^2Logn)
    
            Correct answer is B.

        Q4) In the following C++ function, let n >= m.

        int gcd(int n, int m) {
            if (n%m ==0) return m;
            if (n < m) swap(n, m);
            while (m > 0) {
                n = n%m;
                swap(n, m);
            }
            return n;
        }
        What is the time complexity of the above function assuming n > m?

        A) Θ(logn)
        B) Ω(n)
        C) Θ(loglogn)
        D) Θ(sqrt(n))
    
            Correct answer is A. This question shows swap, addition, comparison operations and all take constant time as well as bitset operations.
        
        Q5) What is the space complexity of the following problem?

        int a = 0, b = 0;
        for (i = 0; i < N; i++) {
            a = a + rand();
        }
        for (j = 0; j < M; j++) {
            b = b + rand();
        }
    
        Space complexity is O(1). rand() takes no extra space.
